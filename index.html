O que é The Tank?

The Tank é um jogo em versão beta desenvolvido com intuito de aprendizado, usando a ferramenta PyGame, com base de desenvolvimento a partir das aulas de LP1 e P1 da universidade federal de alagoas, onde a linguagem python é abordada como instrumento de trabalho das duas matérias.

O jogo tem um objetivo bem simples, é apenas uma "guerra" de tanques que estão presos em um determinado e limitado campo de batalha. Existe níveis de dificuldades onde a cada nível a quantidade de tanques aumenta. O jogo não tem níveis limitados, logo são infinitos níveis e o objetivo é conseguir o maior record possível, essa falta de limitação é proposital por outro motivo também, pois o mesmo é baseado na seguinte historia:

Um soldado muito perverso líder das tropas de tanques de guerras morre em uma das batalhas, ele acaba indo para o inferno mas não sabe, ele continua achando que esta na guerra, porem a guerra nunca acaba, além disso ele esta sempre sozinho lutando contra diversos inimigos, fazendo com que se sinta frustrado e perturbado emocionalmente.

Desenvolvimento

Como citado anteriormente o jogo tem como base a linguagem python e a biblioteca pygame. Para o desenvolvimento desse jogo foram usados alguns conceitos e paradigmas básicos da programação, como programação orienta a objetos, estrutura de dados, CRUD e etc.

Documentação

Todo código está disponivel no repositório @theTank além disso temos também um vídeo explicativo em @theTankOJogo.
